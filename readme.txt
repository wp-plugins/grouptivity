== Description ==

Using Share+ your shared content will be displayed in the Facebook application. So not only can your users view your content in Facebook but it is also ranked by number of times shared.

Grouptivity leverages the news and mini feeds in FB so as your content is read or further shared it is made available to their friends. Making your content viral.

All it requires the publisher to do is downaload the Share+ widget that not only does "email a friend" and Social web sharing, but includes your content in the Facebook app.

* Share+ (with private discussions and forum), 
* Bookmarking
* Cut and Paste
* Tracking and dashboard for your site

== Installation ==

This section describes how to install the plugin and get it working.


1. Download the plugin archive and expand to the wp-content/plugins/ directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. If you have a Grouptivity partner ID, click on the "Share+ configuration" menu item to add it.
